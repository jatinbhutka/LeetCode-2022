```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        result = []
        hashmap = defaultdict(list)
        for s in strs:
            count = [0] * 26
            for c in s:
                count[ord(c)-ord('a')] += 1
            hashmap[tuple(count)].append(s)
        
        return hashmap.values()
```

Time: O(M * N * 26) ---> O(M * N)
Space: O(M * N)
