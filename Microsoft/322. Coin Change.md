```python
class Solution:
    def coinChange(self, coins: List[int], amount: int) -> int:
        
        dp = [amount+1] * (amount+1)
        dp[0] = 0
        
        for a in range(1, amount+1):
            for c in coins:
                if a - c >= 0:
                    # Coin = 4
                    # a = 7
                    # dp[7] = min(dp[7], 1+dp[7-4])
                    #        = min(dp[7], 1 + dp[3])
                    #         = min(8, 2)
                    #         = 2
                    
                    dp[a] = min(dp[a], 1 + dp[a-c])
        
        if dp[amount] == amount+1:
            return -1
        else:
            return dp[amount]     
```

Time: O(amount * len(coin)

Space: O(amount)
