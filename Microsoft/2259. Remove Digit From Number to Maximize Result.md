```python
class Solution:
    def removeDigit(self, number: str, digit: str) -> str:
        
        result = 0
        for i, num in enumerate(number):
            if num == digit:
                result = max(result, int(number[:i]+number[i+1:]))
        return str(result)
        
```
Time: O(N)

Space: O(1)

class Solution:
    def removeDigit(self, number: str, digit: str) -> str:
        num = []
        for i, val in enumerate(number):
            if val == digit:
                num.append(number[:i]+number[i+1:])
        return max(num)
