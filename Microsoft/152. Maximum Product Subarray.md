```python
class Solution:    
    def maxProduct(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """

        min_prod = max_prod = nums[0]
        res = max_prod
        

        for i in range(1,len(nums)):
            curr = nums[i]
            temp = max(curr, max_prod * curr, min_prod * curr)
            min_prod = min(curr, max_prod * curr, min_prod * curr)
            
            max_prod = temp
            res = max(max_prod, res)

        return res
```

Time: O(N)

Space: O(1)
